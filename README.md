## 使用方法

1. 备份游戏根目录下的 `data.i` 文件 (虽然理论上没关系，但保险起见最好也备份下存档)
2. 将 `mod` 文件夹下的内容拷贝至游戏根目录

## 还原方法

1. 删除游戏根目录下的 `/data/system/` 文件夹
2. 将备份的 `data.i` 文件覆盖至游戏根目录
